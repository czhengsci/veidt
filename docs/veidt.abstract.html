class veidt.abstract.Describer¶

Bases: monty.json.MSONable

Base class for a Describer, i.e., something that converts an object to a descriptor, typically a numerical representation useful for machine learning.

describe(obj)¶

Converts an obj to a descriptor.

Parameters:obj – Object
Returns:Descriptor for a structure. Recommended format is a pandas Dataframe object with the column names as intuitive names. For example, a simple site descriptor of the fractional coordinates (this is usually a bad descriptor, so it is just for illustration purposes) can be generated as:
print(pd.DataFrame(s.frac_coords, columns=["a", "b", "c"]))
          a         b         c
0  0.000000  0.000000  0.000000
1  0.750178  0.750178  0.750178
2  0.249822  0.249822  0.249822

Pandas dataframes can be dumped to a variety of formats (json, csv, etc.) easily. Note that a dataframe should be used even if you have only one line, i.e., do not use Series objects unless you know what you are doing.
describe_all(objs)¶

Convenience method to convert a list of objects to a list of descriptors. Default implementation simply loops a call to describe, but in some instances, a batch implementation may be more efficient.

Parameters:objs – List of objects
Returns:Concatenated descriptions for all objects. Recommended format is a pandas DataFrame. Default implement returns a list of descriptors generated by a loop call to describe for each obj.
class veidt.abstract.Model¶

Bases: monty.json.MSONable

Abstract Base class for a Model. Basically, it usually wraps around a deep learning package, e.g., the Sequential Model in Keras, but provides for transparent conversion of arbitrary input and outputs.
